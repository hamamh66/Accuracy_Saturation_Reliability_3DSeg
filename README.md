# When Accuracy Saturates, Reliability Leads

Official repository for the paper on staged evaluation of uncertainty
estimators for 3D pancreas segmentation.

## Contents

-   `Pancreas_Compare.ipynb`
-   `Reliability_Eval.ipynb`
-   `Pred_Disagree.ipynb`
-   `Reliability_Stats.ipynb`

## Summary

This repository contains notebooks and reproducibility artifacts for: -
Multi-seed segmentation comparison - Reliability evaluation -
Uncertainty estimation - Calibration analysis - Statistical analysis

## Dataset

Experiments use the Medical Segmentation Decathlon Task07 (Pancreas)
dataset.
